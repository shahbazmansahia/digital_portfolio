Project Summary

Fake art exposed was a game jam project created for the HuggingFace Open Source Game Jam in 2023. This project utilized a bleeding-edge model called 'LEDITS' which was undergoing a soft release when the game jam was occuring. I implemented batch scripting and automated image-to-image generation using custom prompts that we wished to implement on a dataset of 1000 public domain artworks. We managed to generate a total of 30,000 images from which curated our game assets. This was all done within a span of 3 days. In addition to the code, I also generated the cat assets (AristoCats) using stable diffusion.
This project intended to question what is and isn't 'True Art' and wished to contribute to the ongoing discussions about generative AI artwork.
The game was tied with the winner of the game jam in engagement metrics.

The game can be experienced on itch.io here.


NOTE: The code exists in jupyter notebook format. The notebook executes a gradio app which has been tweaked to suit our purpose. All code written by me is marked in explicit code blocks labelled "CUSTOM CODE" in comments, and in notebook headings as well.
